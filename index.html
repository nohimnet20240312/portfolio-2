Robotics
I designed and built an autonomous obstacle-avoiding robot featuring a custom PCB that integrates the RP2040 microcontroller running MicroPython. The PCB neatly consolidates motor drivers, ultrasonic distance sensors, and a power regulation circuit, enabling the robot to detect and navigate around obstacles in real time. By leveraging MicroPython’s simplicity, I implemented responsive sensor polling and motor control routines, resulting in a compact, efficient platform that demonstrates both hardware design and embedded software proficiency.
I developed a custom seven-segment digital clock using a MAX7219 driver IC to control four 7-segment displays, all orchestrated by a Raspberry Pi Pico running MicroPython. The PCB I designed integrates the MAX7219’s DIN, CLK, and CS lines to the Pico’s GPIO pins (using software SPI), along with a DS3231 real-time clock module for accurate timekeeping and a simple user interface of two push-buttons for setting hours and minutes. The MicroPython firmware reads the RTC, formats the hour and minute values, and sends multiplexed segment data to the MAX7219, updating the display every second. Additional features include adjustable brightness via the MAX7219’s intensity register and a low-power sleep mode that blanks the display when inactive.
